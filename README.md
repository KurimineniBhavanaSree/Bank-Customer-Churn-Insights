Bank Customer Churn Insights
📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes customer churn patterns in the banking sector. The dashboard helps understand customer behavior and identify factors that influence customers leaving the bank.

By analyzing demographic, financial, and activity-related attributes, the dashboard provides insights that can help banks improve customer retention strategies and decision-making.

🎯 Objectives

Analyze patterns of customer churn in the banking sector.

Identify factors influencing customers to leave the bank.

Compare churn behavior across gender, credit score, and account activity.

Provide insights that support customer retention strategies.

🛠 Tools & Technologies Used

Power BI Desktop – Data visualization and dashboard development

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – Creating calculated measures and KPIs

CSV Dataset – Bank Customer Churn dataset

📂 Dataset Details

The dataset contains customer information related to banking services, including:

Customer ID

Age

Gender

Geography (Country)

Credit Score

Tenure (Years with Bank)

Account Balance

Number of Products

Credit Card Ownership

Active Member Status

Estimated Salary

Customer Exit Status (Churn)

These attributes are used to analyze patterns and trends related to customer churn.

📊 Key Metrics & KPIs

The dashboard calculates and displays important business metrics such as:

Total Customers

Active Customers

Inactive Customers

Credit Card Holders

Non-Credit Card Holders

Lost Customers (Churned Customers)

Retained Customers

These KPIs provide a quick overview of the overall customer base and churn distribution.

📈 Dashboard Features

The Power BI dashboard includes several interactive visualizations:

Customer by Year and Activity Category – Shows active vs inactive customer trends.

Monthly Churn Trend Analysis – Displays churn patterns over different months.

Churn by Credit Card Category – Compares churn between credit card holders and non-holders.

Churn by Credit Score – Identifies churn distribution across credit score ranges.

Churn by Gender – Analyzes churn behavior across male and female customers.

Interactive Filters – Users can filter data by year, month, gender, geography, and exit category.

📊 Key Insights

Some insights obtained from the dashboard include:

The dataset contains 10,000 customers.

2,037 customers have exited the bank, while 7,963 customers remain active.

Customer churn varies across credit score categories and account activity levels.

Differences in churn behavior can be observed across gender and credit card ownership.

These insights help identify customer groups that are more likely to leave the bank.

🚀 Impact

This dashboard helps banking organizations:

Identify high-risk customer segments.

Understand patterns influencing customer churn.

Develop data-driven customer retention strategies.

Improve business decision-making using data insights.
